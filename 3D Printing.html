WHAT IS 3D PRINTING

The second module I learned in my fablab journey was 3D Printing. 3D printing, also called Additive Manufacturing, is a process of creating three-dimensional solid objects from a digital file. Unlike traditional manufacturing, where material is removed (subtractive), in 3D printing, objects are built layer by layer. This modern technology is transforming industries like healthcare, aerospace, education, and product design.


Day-1:

- Explore types of 3D printers (FDM, SLA, SLS).
- Understand key terms: filament, nozzle, bed, slicer, STL file.


DAY-2:




3D Printing Workflow*


- Overview of the workflow: Design → Slice → Print → Post-process.
- Introduction to slicing software (e.g., Cura, PrusaSlicer).
- Learn how layer height, infill, and supports affect the print.
In this module, I learned in detail about the 3D printer machine and its important parts:
Extruder – Feeds and melts the filament.
Nozzle – Deposits melted filament layer by layer.
Filament Spool – Supplies the raw material (PLA/ABS).
Heated Bed – Provides a flat, warm surface for the print to stick properly.
Stepper Motors – Move the nozzle and bed in X, Y, and Z directions.
Control Panel & Software – Helps to import files, set parameters, and monitor progress.

I also understood the working principle:

1. A 3D model is created in CAD software (like Fusion 360).
2. The file is converted into a printable format (STL/OBJ).
3. Slicing software converts it into G-code which the printer understands.
4. The printer builds the object layer by layer until the model is complete.
